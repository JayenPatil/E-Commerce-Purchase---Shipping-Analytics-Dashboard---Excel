# 📊 E-commerce Purchase & Shipping Analytics Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive Microsoft Excel dashboard designed to analyze e-commerce sales performance. It helps track sales, profit, customer behavior, product performance, and regional trends using Pivot Tables, Pivot Charts, Slicers, and KPI cards.

---

## 🎯 Objectives

- Analyze overall sales performance
- Track profit and order trends
- Identify top-performing products
- Monitor regional sales
- Improve business decision-making using interactive reports

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

---

## 📈 Dashboard KPIs

- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin

---

## 📊 Dashboard Features

- Interactive Slicers
- Sales Trend Analysis
- Regional Performance
- Category & Sub-Category Analysis
- Top Products
- Customer Insights
- Dynamic KPI Cards
